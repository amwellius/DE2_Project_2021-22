\doxysubsection{Detailed Description}
GPIO library for AVR-\/\+GCC. 


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include "{}\mbox{\hyperlink{gpio_8h}{gpio.h}}"{}} }

\end{DoxyCode}


The library contains functions for controlling AVRs\textquotesingle{} gpio pin(s).

\begin{DoxyNote}{Note}
Based on AVR Libc Reference Manual. Tested on ATmega328P (Arduino Uno), 16 MHz, AVR 8-\/bit Toolchain 3.\+6.\+2. 
\end{DoxyNote}
