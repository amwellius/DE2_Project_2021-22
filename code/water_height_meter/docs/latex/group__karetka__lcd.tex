LCD Nokia 5110 library for AVR-\/\+GCC. 


\begin{DoxyCode}{0}

\end{DoxyCode}


\begin{DoxyNote}{Note}
to modify this code to work with any pinout, modify pin definitions in \mbox{\hyperlink{nokia__5110__lcd_8h}{nokia\+\_\+5110\+\_\+lcd.\+h}}
\end{DoxyNote}
The library contains functions for controlling Nokia 5110 LCD screen via software SPI. With small adjustments it should work with any micrcontroller.
